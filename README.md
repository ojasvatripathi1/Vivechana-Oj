# Vivechana-Oj

Vivechana (विवेचना) is a premium Flutter platform bridging Hindi literature and modern journalism. It functions as a news aggregator, digital magazine, and collaborative writer's space. Powered by Firebase, it offers curated RSS feeds, native PDF viewing, and a secure publishing workflow for independent creators to share poetry, stories, and ideas.

## Features
- **News Aggregator**: Real-time Hindi news from multiple RSS sources.
- **Digital Magazine**: Integrated native PDF reader for monthly editions.
- **Collaborative Writing**: Platform for independent creators to publish poetry and articles.
- **Admin Dashboard**: Secure workflow for content moderation and publishing.
